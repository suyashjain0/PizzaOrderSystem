PIZZA ORDER SYSTEM

The objective of this Java project is to manage the details of Order, Payment, Menu, Customer.
Project is built at both Administrative and Customer end. Admin is guaranteed access to database for 
updating new pizzas and corresponding rates. Customer will be provided with register and login service.
Customer can order different pizza and corresponding bill would be generated for the same. The purpose of this project is to 
build a program to reduce the manual work and ease the ordering system through the application.
